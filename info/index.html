About Payoneとは

イベントのグッズ販売所や飲食店などの商品をオンライン上で事前に「注文」・「決済」することで、
並ばなくてもすぐに受取ることができる新しいモバイルオーダーサービスです。

  • Point 01

    欲しい商品を
    スムーズに手に入れられる

    Payoneでは、イベント開催の数日前から、またイベント当日においても商品の注文・決済をすることができるため、長時間列に並ぶことなく欲しい商品を手に入れることができます。

  • Point 02

    会場での会計の手間が省ける

    事前に決済をするため、会場での現金でのやり取りが発生せずお客様の負担を減らすことができます。

  • Point 03

    入会費や利用料などは
    一切不要

    入会費やサービス利用料などは一切かかりません。何度でもいろんなイベントでPayoneをご利用いただけます。

Flow ご利用の流れ

使い方はとても簡単。以下の4つのステップですぐにご購入＆お受取りいただけます。

  • 商品の選択

    参加予定のイベントページから、欲しい商品を選択して、カートに入れます。

  • ご購入の手続き

    カートから商品をご確認の上、決済を行ってください。
    ※購入の際には、会員登録が必要です。

  • 注文完了

    決済が完了するとQRコードが発行されます。
    ※商品を受取る際に必要です。

  • 商品の受取り

    受取り窓口でQRコードを提示して、商品を受取ってください。

Q&A よくあるご質問

Payoneについて

  • Q

    サービス利用料はかかりますか？

    A全て無料でご利用いただけます。
    ※通信料はお客様のご負担となりますので予めご了承ください。

  • Q

    payone（ペイワン）はどんな端末から利用が可能ですか？

    Aスマートフォン端末（iPhone、Android）からご利用いただけます。

ご注文について

  • Q

    表示されている価格は税込みでしょうか。

    Aすべて税込価格にて表記しています。

  • Q

    商品購入の途中で接続が中断された場合はどうすれば良いですか？

    A恐れ入りますが、再度初めからご注文をお願いいたします。
    決済方法を選択後、「注文完了」画面に変わるまではご注文が完了いたしませんのでご了承ください。
    決済方法を選択後、エラーになった場合は、画面下部アカウントアイコンから注文履歴をご確認ください。

  • Q

    注文した内容はどのように確認すれば良いですか？

    A画面下部アカウントアイコンから「注履履歴」をご確認ください。

  • Q

    エラーが発生したのですがどうすれば良いですか？

    Aこちらからお問い合わせください。

  • Q

    payone（ペイワン）からのメールが届かないのですが。

    A迷惑メールフォルダに振り分けられている場合や、ご登録のメールアドレスの入力間違いによりメールが届かない場合がございます。恐れ入りますが、再度ご確認ください。

  • Q

    注文のキャンセルをするにはどうすれば良いですか？

    Aご注文のキャンセルは承ることができかねます。恐れ入りますが予め内容をよくご確認の上、ご注文をお願いいたします。

商品の受取りについて

  • Q

    商品受取りのQRコードはどこにありますか。

    A画面下部アカウントアイコンから注文履歴をご確認いただき、注文一覧から該当の注文を開いていただきますと、確認できます。

会員登録・変更について

  • Q

    登録内容（ユーザー名・メールアドレス・パスワード）を変更するにはどうすれば良いですか？

    A画面下部アカウントアイコンよりアカウントページからご変更いただけます。
